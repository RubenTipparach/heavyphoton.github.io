Help & Legend

How to use:
  • Range is automatically calculated on load and when parameters change
  • Click/tap on any point to see if the boat can reach it
  • If reachable, the path shows how the boat gets there
  • The boat orientation at the destination is shown
  • Set initial velocities to simulate a moving start
  • Desktop: Use mouse wheel to zoom in/out
  • Desktop: Right-click+drag to pan the view
  • Mobile: Tap and hold to pan the view
  • Mobile: Pinch with two fingers to zoom
